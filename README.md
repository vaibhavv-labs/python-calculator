## 🎯 Project Overview

This is my **first Python project** — a fully functional desktop calculator built using Python's built-in `tkinter` library. It features a dark luxury UI, smooth hover effects, keyboard shortcuts, and a live calculation history panel.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🌙 **Dark Theme UI** | Deep navy dark theme with styled buttons |
| ⌨️ **Keyboard Support** | Type numbers and operators directly from keyboard |
| 📜 **History Panel** | Shows last 3 calculations in the display |
| ➕➖✖️➗ **Full Arithmetic** | Add, subtract, multiply, divide |
| 💯 **Percentage & Sign** | % and +/− toggle buttons |
| 🛡️ **Error Handling** | Safe division by zero handling |

---

## 🖥️ How It Works

```
  ⌨️  User Input (Button Click or Keyboard)
           │
           ▼
  🧠 Calculator Class (Pure Logic)
  (stores current, previous, operator)
           │
           ▼
  🎨 CalculatorApp Class (Tkinter UI)
  (renders display, buttons, history)
           │
           ├── +  ──▶ Addition
           ├── −  ──▶ Subtraction
           ├── ×  ──▶ Multiplication
           └── ÷  ──▶ Division (with zero-check)
                           │
                           ▼
                  📜 Result + History Updated
```

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/YOURUSERNAME/python-calculator.git
cd python-calculator
```

### 2. Run the app
```bash
python calculator.py
```

> ✅ No pip install needed — tkinter comes built-in with Python!

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `0–9` | Input numbers |
| `+ - * /` | Operators |
| `Enter` | Calculate (=) |
| `Escape` | Clear (AC) |
| `Backspace` | Delete last digit |
| `.` | Decimal point |
| `%` | Percentage |
